# Reports

Generated analysis reports and project assessments.

## Structure

Reports are organized by date in ISO format:

```
reports/
└── YYYY/
    └── MM/
        ├── YYYY-MM-DD-analysis.md
        └── YYYY-MM-DD-concerns.md
```

## Generating Reports

Reports are automatically generated by the analysis system:

```bash
# Run analysis and generate report
./tools/analysis/run-analysis.sh
```

See [Analysis System](../tools/analysis/) for more details.

## Report Types

### Analysis Reports

Comprehensive repository health assessment covering:

- Architecture evaluation
- Configuration management
- Security posture
- Operational readiness

### Concerns & Action Plans

Detailed issue identification with:

- Prioritized concerns
- Actionable recommendations
- Implementation timeline
- Risk assessment
