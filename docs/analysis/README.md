# Analysis System

Comprehensive repository analysis framework for periodic health checks.

## Quick Start

```bash
./scripts/analysis/run-analysis.sh
```

## Documentation

- **INDEX.md** - Navigation guide to all documentation
- **QUICK-REF.md** - One-page cheat sheet
- **SYSTEM-GUIDE.md** - Complete user guide
- **SETUP.md** - Setup and installation details

## Generated Reports

Reports are stored in `../reports/YYYY/MM/` with format:
- `MMDD-ANALYSIS.md` - Comprehensive assessment
- `MMDD-CONCERNS_AND_ACTION_PLAN.md` - Actionable items

See [ANALYSIS-QUICK-REF.md](ANALYSIS-QUICK-REF.md) for commands and workflows.
